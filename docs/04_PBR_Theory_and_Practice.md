# 基于物理的渲染 (Physically Based Rendering - PBR)

## 1. 核心理论：重新理解光与物质
传统的渲染（如 Blinn-Phong 模型）基于经验法则来伪造光照，而 PBR 则试图通过物理定律（尤其是光学和微观几何学）在数学上近似真实世界。这带来了惊人的逼真度，以及资产在不同关卡光照下都能“看起来对”的普适性。

PBR 依赖于三个极其核心的物理法则：
### A. 能量守恒 (Energy Conservation)
出射光的能量绝不能超过入射光的能量（除非物体自身发光）。光线击中表面后，会被分为**反射光 (Specular)** 和 **折射光 (Diffuse/Refracted)**。反射得越多，折射/漫反射的必然越少。
在金属工作流中，完全的金属 (Metallic = 1.0) 没有漫反射，它把所有的光都反射了；而绝缘体 (塑料、布料) 则二者皆有。

### B. 微表面理论 (Microfacet Theory)
自然界中没有绝对平坦的表面。在微观看来，任何粗糙表面都是由成千上万个微小的平面（微表面）组成的。
- 如果微表面排列整齐（Roughness 低），光线会朝着同一个方向反射，形成清晰的高光倒影（如镜子）。
- 如果微表面杂乱无章（Roughness 高），光线会被散射到四面八方，形成模糊或完全不存在的高光（如粉笔）。

### C. 菲涅尔方程 (Fresnel Reflectance)
当你垂直低头看水面时，你能看穿水底（低反射）；但当你几乎平行于水面看向远方时，水面像镜子一样反射着天空（高反射）。对于任何物体，**掠射角 (Grazing Angle)** 下的反射率总会急剧增加。

## 2. 核心渲染参数 (金属工作流)

由于 PBR 的推导被统一，美术人员只需要提供以下几张极其符合直觉的贴图（或单值参数）：

1. **Albedo / Base Color (基础颜色)**
   - 对于非金属（绝缘体）：它是表面由于散射而呈现的真正颜色。
   - 对于金属：由于金属没有漫反射，Albedo 代表的是它的**高光反射颜色**（比如金子是黄色的高光，铁是银灰色的高光）。里面绝不应该烤入阴影。
   
2. **Roughness (粗糙度)**
   - 0.0 代表绝对光滑（镜面）。
   - 1.0 代表绝对粗糙（哑光表面）。
   
3. **Metallic (金属度)**
   - 0.0 为绝缘体（不导电物质如木头，石头，塑料等）。
   - 1.0 为纯金属。
   - 在真实物理中，极少有中间值；但在游戏中，0.2 ~ 0.8 的中间值常被用来表示生锈、积满灰尘或半侵蚀状态下的金属。
   
4. **Normal (法线贴图)** / **AO (环境光遮蔽)**
   - 用来在不增加多边形的情况下提供宏观维度的凹凸细节与微小缝隙里的自阴影屏蔽。

## 3. 在 Tumbler 引擎中的代码落地

在我们的 Fragment Shader (`pbr.frag`) 中，我们实现了业界公认的 **Cook-Torrance BRDF**（双向反射分布函数）。它构成了当今所有 3A 引擎光照的基础：

```math
f(l, v) = \frac{D * F * G}{4 (\omega_o \cdot n)(\omega_i \cdot n)}
```

在我们 Vulkan 侧的实现对应关系：
- **D (Distribution GGX)**：法线分布函数，根据 `Roughness` 决定有多少微表面的法线刚好朝向能把光反射进相机的方向（决定高光面积的集中度）。
- **F (Fresnel Schlick 近似)**：菲涅尔项，根据视角与法线的夹角动态增加边缘的高光反射率。对于金属，F0 基础值直接取自 Albedo。
- **G (Geometry Smith)**：几何遮蔽函数，计算微表面之间互相遮蔽阻挡光线的概率（随 `Roughness` 增加而增加），保证高光不会过度亮盲。

最终，我们将漫反射 (Lambert) 与 高光反射 (Cook-Torrance) 通过能量守恒公式相加，与传入的 `CPointLight` 的衰减系数与颜色相乘，最后经过 HDR 调色 Tone Mapping（如 ACES）和伽马校正 (Gamma Correction)，输出最终的像素。
